# Simple Spring Boot CRUD Demo Project

This is a simple Spring Boot project created for educational purposes. The project demonstrates basic CRUD (Create, Read, Update, Delete) operations with MySQL database connectivity.

## Features

- **CRUD Operations:** Perform Create, Read, Update, and Delete operations on a "Student" entity.
- **MySQL Database:** Utilizes MySQL database for storing and retrieving data.
- **Spring Data JPA:** Uses Spring Data JPA for simplifying database operations.
- **Educational Purpose:** Intended to learn Spring Boot, Spring Data JPA, and MySQL database connectivity.

## Prerequisites

- [Java Development Kit (JDK)](https://adoptopenjdk.net/) installed.
- [Wampserver or any other MySQL database server](https://www.wampserver.com/en/) installed and running.
- [Spring Boot](https://start.spring.io/) development environment set up.
